04 · Border, Margin & Padding

Box model lebih detail — ukuran & jarak antar elemen

📦 Box Model Layers

Setiap elemen HTML memiliki 4 layer: margin, border, padding, dan content (dari luar ke dalam).

MARGIN (outermost)
BORDER
PADDING
CONTENT

outside → in: MARGIN → BORDER → PADDING → CONTENT

↔️ MARGIN

Ruang TRANSPARAN di luar elemen (di luar border). Digunakan untuk memberi jarak antar elemen.

margin: 20px; /* semua sisi */ margin: 10px 20px; /* top/bottom 10px, left/right 20px */ margin: 10px 20px 30px 40px; /* top right bottom left */ margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px;

Margin Examples:

margin: 20px
margin-top: 30px
margin: 40px

💡 Margin Collapse:

Margin vertikal antar elemen block akan collapse (tidak ditambah). Contoh: div dengan margin-bottom 20px di atas div dengan margin-top 30px = total margin 30px (bukan 50px).

🖼️ BORDER

Garis TERLIHAT yang membatasi elemen. Bagian dari lebar/tinggi element jika box-sizing: border-box.

border: 2px solid #58A6FF; border: [width] [style] [color]; border-width: 2px; border-style: solid | dashed | dotted | double | groove | ridge | inset | outset; border-color: #58A6FF; border-top: 2px solid red; border-right: 2px dashed green; border-bottom: 2px dotted blue; border-left: 2px double orange;

Border Examples:

solid
dashed
dotted
double

Border Radius:

border-radius: 5px
border-radius: 10px
border-radius: 20px
border-radius: 50%

📐 PADDING

Ruang DALAM elemen (di dalam border, di luar content). Digunakan untuk memberi ruang di dalam elemen.

padding: 20px; /* semua sisi */ padding: 10px 20px; /* top/bottom 10px, left/right 20px */ padding: 10px 20px 30px 40px; /* top right bottom left */ padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px;

Padding Examples:

padding: 10px
padding: 20px
padding: 30px

💡 Difference: Padding vs Margin

  • Padding: Ruang dalam (inside border) - background-color akan terisi
  • Margin: Ruang luar (outside border) - transparent, tidak terisi

📏 WIDTH & HEIGHT

Ukuran konten elemen. Dapat dipengaruhi oleh padding dan border tergantung box-sizing.

width: 300px; /* fixed width */ width: 50%; /* percentage of parent */ width: auto; /* default */ width: 100vw; /* viewport width */ height: 200px; /* fixed height */ height: auto; /* default */ height: 100vh; /* full viewport height */ max-width: 100%; /* responsive max width */ min-width: 100px;

Width & Height Examples:

300px × 100px (fixed)
50% × 100px
auto × auto

📦 BOX SIZING

Menentukan apakah width/height hanya content, atau termasuk padding dan border.

box-sizing: content-box; /* default */ width: 300px → hanya content, total lebih besar box-sizing: border-box; /* include border & padding */ width: 300px → include padding & border, lebih predictable

Box Sizing Comparison:

content-box (default)

300px width (hanya content)

Actual width: 300px + 20px padding × 2 + 2px border × 2 = 344px

border-box ✓ recommended

300px width (include all)

Actual width: exactly 300px

💡 Best Practice:

Gunakan * { box-sizing: border-box; } di awal CSS file Anda untuk membuat sizing lebih predictable!

📚 Box Model Properties Reference

Property Values Deskripsi
margin px, %, em, auto Ruang luar elemen
border width style color Garis pembatas
padding px, %, em Ruang dalam elemen
width px, %, auto, vw Lebar konten
height px, %, auto, vh Tinggi konten
box-sizing content-box, border-box Include padding/border dalam width/height?
