08 · Colors & Contrast

Format warna, pemilihan palet & aksesibilitas

🎨 Color Formats

Cara berbeda untuk mendefinisikan warna di CSS.

/* HEX - Paling umum */ color: #FF0000; /* Red */ color: #58A6FF; /* Blue */ color: #f0f; /* Shorthand (3 digits) */ /* RGB - Red Green Blue */ color: rgb(255, 0, 0); /* Red */ color: rgb(88, 166, 255); /* Blue */ /* RGBA - RGB dengan opacity */ color: rgba(255, 0, 0, 1); /* Opaque red */ color: rgba(255, 0, 0, 0.5); /* 50% transparent red */ /* HSL - Hue Saturation Lightness */ color: hsl(0, 100%, 50%); /* Red */ color: hsl(212, 100%, 67%); /* Blue */

♿ Color Contrast & Accessibility

WCAG 2.1: Contrast minimum AA = 4.5:1 untuk teks normal, 3:1 untuk teks besar.

Contrast Ratio: Perbandingan antara brightness teks dan background.

Levels:

  • AAA: 7:1 atau lebih (enhanced contrast)
  • AA: 4.5:1 (standard, recommended)
  • Fail: Kurang dari 3:1
White on Black
Contrast: 21:1 (AAA)
Blue on Dark Gray
Contrast: 4.8:1 (AA)
Green on Dark
Contrast: 7.2:1 (AAA)
Gray on Gray
Contrast: 1.5:1 (FAIL)

✅ Accessibility Tips:

  • Gunakan online contrast checker (e.g., WebAIM, Contrast Checker)
  • Hindari hanya mengandalkan warna (gunakan text labels juga)
  • Test dengan color blindness simulator
  • Pastikan text selalu readable di semua background

Tips Memilih Warna yang Accessible:

  • ✓ Gunakan light text pada dark background atau sebaliknya
  • ✓ Hindari kombinasi warna yang mudah confusion (e.g., red-green untuk color blind users)
  • ✓ Test contrast menggunakan DevTools atau online tools
  • ✓ Jangan hanya mengandalkan warna untuk membedakan (gunakan icon/label juga)
