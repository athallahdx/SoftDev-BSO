03 · The Holy Grail of Layouting

Flexbox · CSS Grid · Advanced Layouts

📱 FLEXBOX (1D Layout)

Layout satu arah dengan kontrol flow dan alignment yang fleksibel. Sempurna untuk navbar, button group, dan centering.

display: flex; flex-direction: row | column; justify-content: center | space-between | space-around; align-items: center | stretch | flex-start; gap: 1rem;

Flexbox Properties

display: flex Mengaktifkan flexbox layout
flex-direction row (default) | column | row-reverse | column-reverse
justify-content Alignment horizontal (main axis)
align-items Alignment vertikal (cross axis)
gap Jarak antar item
flex-wrap wrap | nowrap (default)
flex Shorthand untuk grow, shrink, basis

✓ Cocok Untuk:

  • Navbar dan navigation menu
  • Button group dan toolbar
  • Form input (vertical/horizontal)
  • Centering elemen
  • Simple card layout

🎯 CSS GRID (2D Layout)

Layout dua arah dengan kontrol presisi terhadap row dan column. Sempurna untuk dashboard dan page layout.

display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 1fr auto; grid-gap: 16px; grid-area: header | main | sidebar | footer;

CSS Grid Properties

display: grid Mengaktifkan grid layout
grid-template-columns Definisikan width setiap column
grid-template-rows Definisikan height setiap row
grid-gap / gap Jarak antar grid items
grid-area Assign item ke named grid area
grid-column / grid-row Span elemen across multiple columns/rows

✓ Cocok Untuk:

  • Dashboard dengan multiple panels
  • Card/catalog layout
  • Page layout (header, nav, content, sidebar, footer)
  • Gallery dengan uniform grid
  • Kompleks 2D layout yang presisi

⚖️ Flexbox vs CSS Grid

Flexbox (1D)

  • Layout satu arah
  • Fokus: flow & alignment
  • Konten lebih fluid
  • Items fleksibel
  • Easier to use

CSS Grid (2D)

  • Layout dua arah
  • Fokus: struktur & posisi
  • Lebih presisi & konsisten
  • Layout terstruktur
  • Powerful untuk kompleks

🎁 BONUS: Table Layout (Legacy)

display: table / table-row / table-cell — fallback untuk browser lama, berguna untuk vertical alignment.

Table layout hanya digunakan sebagai fallback untuk browser yang tidak support Flexbox/Grid. Untuk browser modern, gunakan Flexbox atau Grid!
