01 · Basic Elements in HTML

The building blocks of every web page

📐 HTML STRUCTURE

Struktur wajib setiap halaman web. Setiap halaman dimulai dengan struktur dasar yang sama.

<html> <head> <meta charset="UTF-8"> <title>Judul Halaman</title> </head> <body> <!-- Konten halaman di sini --> </body> </html>
<html>

Root element — wrapper untuk seluruh halaman

<head>

Metadata — title, charset, styles, scripts

<body>

Konten yang terlihat di browser

🔗 LINKS & IMAGES

Navigasi & media visual — elemen penting untuk interaktivitas dan visual halaman.

<a href="https://example.com">Teks Link</a> <img src="image.jpg" alt="Deskripsi Gambar"> <a href="page.html" title="Hover Text">Internal Link</a> <a href="#section">Anchor Link</a>

  • Gunakan title untuk informasi tambahan
  • Gunakan <figure> dan <figcaption> untuk gambar penting

📝 TEXT ELEMENTS

Elemen teks untuk memberikan struktur semantik pada konten halaman.

<h1>Heading 1</h1> <!-- Judul utama --> <h2>Heading 2</h2> <!-- Sub-judul --> <h3>Heading 3</h3> <!-- Sub-sub-judul --> <p>Paragraf teks...</p> <span>Inline text</span> <div>Block container</div>

Ini adalah H1 (Heading 1)

Heading 1 adalah yang terbesar — untuk judul utama halaman

Ini adalah H2 (Heading 2)

Heading 2 untuk sub-judul — tingkat hierarchi di bawah H1

Ini adalah H3 (Heading 3)

Heading 3 untuk sub-sub-judul

Ini adalah paragraph biasa. Digunakan untuk blok teks yang berisi satu ide utama. Paragraf harus selalu dibungkus dengan tag <p> untuk semantic HTML.

Ini adalah contoh inline text dengan span — span tidak membuat baris baru.

💡 Semantic HTML:

  • Gunakan H1 hanya satu kali per halaman (untuk judul utama)
  • Gunakan heading dalam urutan yang benar (H1 → H2 → H3)
  • Paragraf untuk blok teks, span untuk inline text
  • Div untuk container/grouping elemen

📋 LISTS & FORMS

Elemen interaktif untuk user input dan daftar — penting untuk navigasi dan form submission.

<!-- Unordered List --> <ul> <li>Item 1</li> <li>Item 2</li> </ul> <!-- Ordered List --> <ol> <li>Step 1</li> <li>Step 2</li> </ol> <!-- Form --> <form> <input type="text" name="username"> <button type="submit">Submit</button> </form>

Unordered List (<ul>)

  • Item pertama
  • Item kedua
  • Item ketiga
  • Item keempat

Ordered List (<ol>)

  1. Step pertama
  2. Step kedua
  3. Step ketiga
  4. Step keempat

Form Elements (<form>)

💡 Form Best Practices:

  • Selalu gunakan <label> dengan for attribute
  • Gunakan type yang sesuai (text, email, password, dll)
  • Tambahkan placeholder untuk hint kepada user
  • Gunakan required untuk validasi basic HTML

🏷️ SEMANTIC HTML ELEMENTS

Elemen semantik memberikan makna pada struktur halaman — lebih baik untuk SEO dan accessibility.

<header>Header halaman</header> <nav>Navigasi menu</nav> <main>Konten utama</main> <article>Artikel/post</article> <aside>Sidebar</aside> <section>Bagian/topik</section> <footer>Footer halaman</footer>
