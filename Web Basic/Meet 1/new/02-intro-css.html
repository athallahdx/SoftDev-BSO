Memilih elemen dengan class tertentu

margin → border → padding → [ content ]

🔢 CSS Specificity (Prioritas)

CSS Specificity menentukan rule mana yang lebih diprioritaskan ketika ada conflict.

1
Element Selector p { }
10
Class Selector .class { }
100
ID Selector #id { }
1000
Inline Style style=""

Contoh:

/* Specificity: 1 */ p { color: blue; } /* Specificity: 10 — akan override p */ .highlight { color: yellow; } /* Specificity: 100 — akan override .highlight */ #special { color: red; } /* Specificity: 1000 — akan override semuanya */ <p style="color: green;">Teks ini hijau</p>
