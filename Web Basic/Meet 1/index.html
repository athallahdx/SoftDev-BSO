03 · The Holy Grail of Layouting

Flexbox · CSS Grid · Advanced Layouts

Flexbox (1D Layout)

Layout satu arah dengan kontrol flow dan alignment yang fleksibel.

Item 1
Item 2
Item 3
Item 4
display: flex; justify-content: space-around; align-items: center; gap: 1rem;

Cocok untuk: Navbar, Button group, Form, Centering

CSS Grid (2D Layout)

Layout dua arah dengan kontrol presisi terhadap row dan column.

Header
Sidebar
Main Content
Footer
display: grid; grid-template-columns: 200px 1fr; grid-template-rows: auto 1fr auto; gap: 16px;

Cocok untuk: Card/catalog, Dashboard, Page layout, Gallery

Flexbox vs CSS Grid

Aspek Flexbox CSS Grid
Dimensi 1D (satu arah) 2D (row & column)
Fokus Flow & alignment Struktur & posisi
Konten Fluid (ikut isi) Presisi & konsisten
Best For Navbar, buttons, forms Dashboard, gallery, layouts

05 · Z-Index & Stacking Context

Layer ordering — higher z-index = on top

Z-Index: 100
Z-Index: 50
Z-Index: 10

Key Rules

  • ✓ z-index hanya bekerja pada positioned elements (relative, absolute, fixed, sticky)
  • ✓ Stacking context dibuat oleh: position + z-index, opacity < 1, transform, filter
  • ✓ z-index: auto = TIDAK membuat stacking context baru
  • ✓ z-index: 0 = MEMBUAT stacking context baru

Typography Best Practices

Property Rekomendasi Penjelasan
font-size 16px body · 24–48px heading Ukuran yang tepat untuk readability
font-family sans-serif, monospace, serif Pilih yang consistent dengan brand
font-weight 400 normal · 700 bold Contrast visual untuk hierarchy
line-height 1.5–1.8 Space antar baris untuk kenyamanan baca
letter-spacing 0–0.1em Jarak antar karakter (tracking)
text-align left body · center header Alignment yang sesuai konteks
